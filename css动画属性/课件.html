css第二讲

2d 变换

  • transform 变换
    • transform: translate(45px ,12px);
      • 水平位移
      • 垂直位移
    • transform: rotate(-50deg);
      • 旋转角度
      • 正值-顺时针
      • 反之，逆时针
    • transform: scale(1,.5)
      • 水平缩放倍数
      • 垂直缩放倍数
    • transform:skew(0deg,45deg)
      • 水平倾斜角度
      • 垂直倾斜角度
  • matrix(a,b,c,d,e,f)
    • translate(tx,ty) ==== matrix(1,0,0,1,tx,ty);
    • scale(sx,sy) ==== matrix(sx,0,0,sy,0,0);
    • rotate(deg) ==== matrix(cos(deg),sin(deg),-sin(deg),cos(deg),0,0);
    • skew(dx,dy) ==== matrix(1,tan(dy),tan(dx),1,0,0)

过渡

  • transition
    • transition-property 。。。 指定要参与过渡的属性
    • transition-duration 。。。。过渡的时长
    • transition-timing-function 。。。。过渡的速度
    • transition-delay 。。。。。。过渡延迟

3d 变换

  • transform
    • rotateX
    • rotateY
    • rotateZ
    • rotate3d(1,0,0,90deg);
      • x轴比例
      • y轴比例
      • z轴比例
      • 角度
    • translateX(40px) translateY(40px) translateZ(-300px)
    • translateX
    • translateY
    • translateZ
    • translate3d(100px ,100px ,-300px)
- scaleX
- scaleY
- scale3d(1,2,3)

- skewX
- skewY
  • perspective: 21px; 。。。眼睛距离三维物体的距离
  • transform-origin: 物体旋转的起点或者方向
  • perspective-origin : 规定我们看三维物体的角度或者起点
  • transform-style: preserve-3d|flat;
    • preserve-3d 保留子元素的3D变换效果
    • flat ，不保留3d效果，扁平化
  • backface-visibility: visible||hidden;背面的可见性
